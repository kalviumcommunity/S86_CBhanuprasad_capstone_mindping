# 🧠 MindPing – Focus-First Productivity App

MindPing is a productivity app designed to help users **focus on one task at a time**.  
It combines **AI-powered task prioritization**, **distraction-free focus sessions**, and **simple task management** to promote deep work and reduce mental overload.

---

## ✨ Core Idea

Most productivity apps overwhelm users with lists.  
**MindPing solves this by guiding users to focus on the single most important task right now.**

---

## 🚀 Features

### 🔐 Authentication
- Email-based sign up & login
- Persistent user sessions

### 📋 Smart Task Dashboard
- Tasks automatically **sorted by AI priority**
- Clear visibility of what to work on next
- Minimal, clean UI

### ➕ Task Creation
- **Quick Add** – instantly add tasks
- **AI Chat Add** – add tasks using natural language  
  _Example: “Finish math assignment by tomorrow evening”_

### 🎯 Focus Mode (Core Feature)
- Enter distraction-free mode
- Focus timer (Pomodoro-style)
- Works on **one selected task**
- Optional notification blocking

### 📈 Session Summary
- Time spent focusing
- Task progress update
- Options to:
  - Continue focus
  - Take a break
  - Move to next task

### 📊 Progress Tracking (Planned)
- Daily & weekly focus time
- Completed task count
- Productivity insights

---

## 🧭 App Flow

1. Welcome Screen  
2. Sign Up / Log In  
3. Task Dashboard (AI-prioritized)  
4. Add Tasks (Quick Add or AI Chat)  
5. Enter Focus Mode  
6. Complete Focus Session  
7. View Progress & Continue
